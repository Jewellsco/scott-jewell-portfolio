# Scott Jewell — Portfolio

Live site: https://scott-jewell-portfolio-site.vercel.app/

Static site (HTML/CSS/vanilla JS, no build step). Deployed on Vercel.
Pushing to the connected branch auto-deploys.

- `index.html` — portfolio home (contact overlay, work, about, experience)
- `case-*.html` — case studies (CVS, Caregiver, Freebird, Brand GO, BagEquity)
- `*-prototype.html` / `caregiver-redesign.html` — embedded interactive prototypes
- `Scott-Jewell-Resume.html` / `Scott-Jewell-Resume.pdf` — résumé
- `vercel.json` — clean URLs + security headers
